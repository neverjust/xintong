# 认证

## 默认认证

### 功能描述

判断用户是否已经认证过

### 请求说明

> 请求方式：POST<br>
> 请求URL ：/Authenticate/index

### 请求参数

| 字段 | 字段类型 | 字段说明 |
| :--- | -------- | -------- |
| code | string   | wx_index |

### 返回结果

```json  
{
  "data": {
  },
  "code": "200",
  "msg": "SUCCESS"
}
```

### 返回参数

| 字段 | 字段类型 | 字段说明 |
| ---- | -------- | -------- |
|      |          |          |

### 错误状态码

| 状态码 | 说明                              |
| ------ | --------------------------------- |
| 2001   | 已经认证 是学生                   |
| 3002   | 参数不完整                        |
| 2002   | 已经认证 石老师                   |
| 2003   | 未认证                            |
| 2006   | 无法获取openid 连接不到微信服务器 |
